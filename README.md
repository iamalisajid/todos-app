## REACT-CONTACTS-TODO-APP

A react-redux based app for managing todos, contacts. Using following opensource libraries:

* React
* Redux
* React-Router
* React-Redux
* Redux-Persist
* Styled-Components
* Storybooks
* Json-Server

Install the dependencies and devDependencies and start the server.

```sh
$ cd todos-app
$ npm install
$ json-server db.json
$ npm start -s
```
